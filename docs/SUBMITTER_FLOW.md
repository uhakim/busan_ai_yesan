# 회원용 AI 구독료 제출 흐름

## 목적

회원은 관리자용 상세 지출 입력을 사용하지 않고, 본인이 결제한 AI 구독료와 증빙만 간단히 제출한다.

## 화면

`회원용 제출` 탭을 별도로 둔다.

입력 필드:

- 이름
- AI 서비스명
- 결제일
- 결제금액
- 비고
- AI 서비스 영수증(외화영수증)
- 카드사 원화전표

첨부 제한:

- 허용 형식: JPG, PNG, WebP, PDF
- 파일 1개당 최대 10MB
- 제출 1건당 첨부 전체 최대 20MB

제출 시 저장되는 값:

- `category`: `direct`
- `directType`: `ai_subscription`
- `description`: `{AI 서비스명} AI 구독료`
- `itemName`: `AI 구독료`
- `paymentMethod`: `card`

학교명과 동아리명은 제출자 화면에 노출하지 않고 관리자 기본 정보에서 가져온다.

## 관리자 확인

회원이 제출한 건은 기존 지출 내역과 같은 데이터로 저장한다.

- `AI 구독료 현황` 탭의 월별 표에서 회원별 월별 사용액과 회원별 합계 확인
- `AI 구독료 현황` 탭 하단에서 전체 사용액과 사용가능 잔액 확인
- 관리자는 회원별 AI 구독료 가능금액을 설정하고, 가능금액 합계는 전체 AI 구독료 배정액을 넘지 못하게 한다.
- 회원 제출액이 개인 가능금액을 넘는 경우 제출은 저장하되 검토 경고로 표시한다.
- `검토` 탭에서 증빙 누락과 상태 확인
- `검토` 탭에서 `증빙 보기`로 첨부한 영수증/PDF 원본 미리보기를 확인한다.
- 관리자가 회원 통장으로 금액을 지급하면 해당 지출 건을 `지급 완료`로 표시하고, 회원과 관리자가 지급 여부를 함께 확인한다.
- `최종 출력` 탭에서 PDF 출력 대상에 포함

## 월별 집계 기준

월별 현황은 지출일자의 `YYYY-MM` 기준으로 묶는다.

1년 구독료처럼 한 번에 결제한 금액은 우선 결제월에 전액 반영한다. 이후 필요하면 이용 기간 기준 월별 배분 옵션을 추가한다.
