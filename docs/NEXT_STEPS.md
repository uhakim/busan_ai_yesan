# 다음 작업

## 1차 프로토타입

`index.html` 단일 파일로 로컬 웹앱을 만든다.

필수 기능:

- 지출 내역 추가/수정/삭제
- 연구활동비, 직접성 경비, 업무협의회비 선택
- 지출일자, 금액, 사용내용 입력
- 영수증 파일 첨부와 미리보기
- 항목별 합계와 한도 경고
- JSON 저장/불러오기
- 인쇄용 정산서 화면

## 2차 프로토타입

- 제출자별 입력 링크 느낌의 화면 분리
- 관리자 검토 상태 추가
- 증빙번호 자동 생성
- 누락 증빙 점검표
- 출력 화면을 정산서 PPT 양식 순서에 맞춤

## 3차 프로토타입

- XLSX 자동 생성
- PPTX 양식 자동 채우기
- Google Drive 업로드 연동
- Google Form 제출 데이터 불러오기

## 결정할 것

- 데이터 저장을 브라우저 localStorage로 할지 JSON 파일 기반으로 할지
- 선생님들이 직접 입력하게 할지, 유하님이 받은 자료를 대신 입력하게 할지
- 최종 제출물이 PDF면 충분한지, PPTX 원본 양식까지 자동 생성해야 하는지
- 실제 운영할 때 외부 배포가 필요한지
