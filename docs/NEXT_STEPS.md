`index.html` 단일 파일로 로컬 웹앱을 만든다.

필수 기능:

- 연구활동비, 연수 운영비, 직접성 경비, 업무협의회비 선택
- 지출일자, 금액, 사용내용 입력
- 영수증 파일 첨부와 미리보기
- 항목별 합계와 한도 경고
- JSON 저장/불러오기
- 인쇄용 정산서 화면

## 2차 프로토타입

- `docs/SUBMITTER_FLOW.md` 기준으로 회원용 AI 구독료 간단 제출 탭 추가
- 학교명과 동아리명은 관리자 기본 정보에서 한 번만 입력하고 제출자 화면에서는 숨김
- 제출자용 JSON 다운로드와 관리자용 제출 JSON 합치기 추가
- 직접성 경비 세부 유형에 AI 구독료를 추가하고 월별 표에서 회원별 합계 확인
- 관리자 기본 정보에서 총예산, 항목별 예산, 직접성 경비 내 AI 구독료 배정액 조절
- AI 구독료 현황을 별도 탭으로 분리하고 결제월 기준 회원별 월별 사용 현황 표와 하단 전체 합계/잔액 확인
- 회원별 AI 구독료 가능금액 배정, 개인별 잔액 확인, 초과 제출 경고 표시
- 검토 탭에서 첨부 증빙 원본 미리보기와 다운로드 지원
- 지출 건별 지급 완료 표시, 미지급액/실지급 잔액 확인
- 원본 PPTX 양식 기준 붙여넣기용 표 생성
- 제출자별 입력 링크 느낌의 화면 분리
- 관리자 검토 상태 추가
- 증빙번호 자동 생성
- 누락 증빙 점검표
- 출력 화면을 정산서 PPT 양식 순서에 맞춤
