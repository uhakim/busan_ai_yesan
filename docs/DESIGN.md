# AI 디지털 교사 동아리 정산 도우미 설계

## 목적

AI 디지털 교사 동아리 정산 업무에서 선생님별 영수증, 지출 내역, 증빙 파일을 모으고 검토한 뒤 정산서 양식에 맞춰 최종 출력물을 만드는 도구다.

## 기본 방향

- CORE LAB 사이트의 사업비 탭 UX를 참고하되, 우리 동아리 정산은 훨씬 단순하게 만든다.
- 예산 항목은 원본 PPTX 제출 양식에 맞춰 4개로 둔다.
  - 연구활동비
  - 연수 운영비
  - 직접성 경비
  - 업무협의회비
- 기본 예산값은 총 2,000,000원, 연구활동비 1,000,000원, 연수 운영비 0원, 직접성 경비 800,000원, 업무협의회비 200,000원이다.
- 관리자는 기본 정보 화면에서 총예산과 항목별 예산을 조정할 수 있다.
- 직접성 경비 안에서 AI 구독료 배정액을 별도로 정하고 사용가능 잔액을 확인한다.

## 사용자

### 제출자

동아리 구성원 선생님.

해야 할 일:

- 지출 정보 입력
- 영수증 또는 계좌이체 확인증 첨부
- 해외 AI 구독이면 원화 카드 전표 첨부
- 업무협의회비면 협의록 또는 참석 증빙 첨부

### 관리자

유하님 또는 동아리 총무.

해야 할 일:

- 제출 내역 검토
- 누락 증빙 확인
- 항목별 합계 확인
- 회원에게 통장 지급이 완료된 건 표시
- 오류 또는 반려 메모 작성
- 최종 정산서 출력

## 화면 구조

### 1. 대시보드

상단 카드:

- 총 예산
- 사용액
- 잔액
- 사용률
- 누락 증빙 건수
- 지급 완료액
- 미지급액
- 실지급 잔액

항목별 표:

| 항목 | 한도 | 사용액 | 지급완료 | 미지급 | 잔액 | 상태 |
|---|---:|---:|---:|---:|---:|---|
| 연구활동비 | 입력값 | 자동 계산 | 자동 계산 | 자동 계산 | 자동 계산 | 정상/초과 |
| 연수 운영비 | 입력값 | 자동 계산 | 자동 계산 | 자동 계산 | 자동 계산 | 정상/초과 |
| 직접성 경비 | 입력값 | 자동 계산 | 자동 계산 | 자동 계산 | 자동 계산 | 증빙 확인/초과 |
| 업무협의회비 | 입력값 | 자동 계산 | 자동 계산 | 자동 계산 | 자동 계산 | 정상/초과 |

### 2. 회원용 AI 구독료 제출 화면

회원은 상세 정산 항목을 고르지 않고 AI 구독료만 간단히 제출한다.

필수 입력:

- 이름
- AI 서비스명
- 결제일
- 결제금액
- 영수증/구매내역서

선택 입력:

- 비고
- 외화 영수증
- 국내 카드사 원화 전표

저장 방식:

- 예산 항목은 직접성 경비로 자동 저장한다.
- 직접성 경비 세부 유형은 AI 구독료로 자동 저장한다.
- 학교명과 동아리명은 관리자 기본 정보에서 가져오고 제출자 화면에는 노출하지 않는다.

### 3. AI 구독료 현황 화면

- 직접성 경비 중 세부 유형이 AI 구독료인 건만 별도 집계한다.
- 지출일자의 결제월 기준으로 회원별 월별 사용액을 확인한다.
- 월별 표 하단에 전체 합계와 AI 구독료 배정액 기준 사용가능 잔액을 표시한다.
- AI 구독료 배정액은 관리자 기본 정보 화면에서 직접 조정한다.

### 4. 관리자 상세 지출 입력 화면

기본 정보 입력:

- 학교
- 동아리명
- 총 예산
- 연구활동비 예산
- 연수 운영비 예산
- 직접성 경비 예산
- 업무협의회비 예산
- AI 구독료 배정액
- 학교장 성명
- 담당자 연락처
- 이자

필수 입력:

- 제출자 이름
- 학교
- 동아리명
- 예산 항목
- 직접성 경비 세부 유형: AI 구독료 / 도서 / 인쇄비 / 기타 직접성 경비
- 사용 내용
- 지출일자
- 지출 금액
- 결제방법: 카드결제 / 계좌이체 / 기타
- 품명
- 지급처 또는 상호

선택 입력:

- 단위
- 수량
- 단가
- 비고
- 증빙번호 직접 입력

첨부:

- 카드 영수증
- 계좌이체 확인증
- 거래명세서
- 외화 영수증
- 국내 카드사 원화 전표
- 협의록
- 활동 사진

### 5. 관리자 확인 화면

기능:

- 제출자별 필터
- 항목별 필터
- 증빙 누락 필터
- 금액 초과 경고
- 미확인/확인/반려 상태
- 지급 완료/미지급 상태
- 지급 완료일 자동 기록
- 반려 사유 메모
- 증빙 파일 미리보기

### 6. 최종 출력 화면

정산서 양식에 맞춘 인쇄용 화면을 만든다.

출력 섹션:

- 정산서 표지
- 운영비 실적 총괄표
- AI 구독료 회원별·월별 현황
- 연구활동비 지출 세부내역
- 연수 운영비 지출 세부내역
- 직접성 경비 지출 세부내역
- 업무협의회비 지출 세부내역
- 항목별 증빙자료 묶음

출력 방식:

- 1차: 브라우저 인쇄로 PDF 저장
- 2차: XLSX 자동 생성
- 3차: PPTX 양식 자동 채우기

## 검증 규칙

### 공통

- 금액은 0보다 커야 한다.
- 지출일자는 사업 기간 안이어야 한다.
- 사용 내용과 지급처는 비워둘 수 없다.
- 카드결제는 카드 영수증이 필요하다.
- 계좌이체는 이체확인증이 필요하다.
- 간이영수증은 경고한다.
- 자산취득성 물품은 경고한다.

### 연구활동비

- 총액 1,000,000원 초과 시 경고한다.
- 수당류는 피지급자, 지급 기준, 계좌이체 확인증이 필요하다.

### 연수 운영비

- 동아리 운영 목적의 시설 임차, 인쇄 등 운영성 지출만 허용한다.
- 상세 품목·단가가 적힌 지출 내역서, 신용카드 매출전표 또는 세금계산서를 요구한다.

### 직접성 경비

- AI 구독료, 도서, 인쇄비 등 사업 목적 직접 관련 지출만 허용한다.
- 해외 AI 구독이면 외화 영수증 또는 서비스 공식 영수증을 요구하고, 실제 지급액 확인을 위해 국내 카드사 원화 전표를 함께 요구한다.
- 구독 이용 기간이 2026년 12월까지인지 확인한다.

### 업무협의회비

- 총액 200,000원 초과 시 경고한다.
- 식비/다과비만 가능하다고 안내한다.
- 협의록, 참석자, 일시, 장소, 회의 내용, 사진 첨부를 요구한다.
